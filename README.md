# Zombi Game

3D браузерная игра на выживание против зомби в стиле Roblox 2015 года.

## Описание

Вы — выживший в мире, захваченном зомби. Ваша задача — продержаться как можно дольше, отстреливая волны нежити. С каждой волной зомби становится больше, а их типы — опаснее. Зарабатывайте монеты за убийства и покупайте оружие, броню и антидоты в магазине.

## Как запустить

1. Скачайте или клонируйте репозиторий
2. Откройте файл `index.html` в любом современном браузере
3. Игра запустится автоматически

**Никакой установки, сервера или сборки не требуется!**

## Управление

| Клавиша | Действие |
|---------|----------|
| **Стрелки** | Движение |
| **Пробел** | Прыжок |
| **Мышь** | Прицеливание |
| **ЛКМ** | Стрельба |
| **R** | Перезарядка |
| **1-3** | Выбор оружия |
| **B** | Открыть/закрыть магазин |

## Игровые механики

### Система заражения
- При укусе зомби начинается заражение
- У вас есть **2 минуты** чтобы купить антидот
- Каждый последующий укус ускоряет таймер
- Если не успеете — превратитесь в зомби (конец игры)

### Система повреждений конечностей (только дробовик)
- **Выстрел в ногу**: зомби начинает ползти (скорость -70%)
- **Выстрел в руку**: урон зомби снижается на 40%
- **Обе руки отстрелены**: урон зомби снижен на 80%

### Магазин (клавиша B)

| Товар | Цена | Эффект |
|-------|------|--------|
| Броня | 50 монет | +50 к защите |
| AK-47 | 400 монет | Автоматическая винтовка |
| Дробовик | 400 монет | Мощное ближнее оружие |
| Антидот | 100 монет | Лечит заражение |
| Хазмат-костюм | 600 монет | +600 к защите + противогаз |

## Советы

- Цельтесь в голову для максимального урона
- Дробовик отлично подходит для ближнего боя и толпы
- Следите за таймером заражения — антидот спасёт вам жизнь
- Копите на хазмат-костюм для долгого выживания
- Используйте дробовик для обездвиживания опасных зомби (отстреливайте ноги)
